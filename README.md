CLI for [rainywebpack](https://github.com/geekrainy/rainywebpack).

## Getting Started

```shell
# Install
$ yarn global add rw-cli

# Create rainywebpack project
$ rw new demo

# Start rainywebpack
$ cd demo
$ yarn start
```
